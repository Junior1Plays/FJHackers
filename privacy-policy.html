Política de Privacidade


Quando visto "nós" e/ou "nosso", refere-se a FJHackers em geral.

Coleta e uso de dados

Nós podemos coletar dados a fim de propósitos diferentes, os dados coletados podem vir de serviços externos.

Tipos de dados coletados

Dados pessoais

Durante o uso do aplicativo, podemos coletar dados como Endereço IP e informações fornecidas pelo usuário.

Uso de dados

Nós podemos usar os dados fornecidos para identificar o usuário.

Segurança de dados

Os dados coletados são criptografados durante o caminho dos nossos servidores, garantindo a segurança dos dados.
